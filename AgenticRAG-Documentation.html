1. Overview

This guide describes how to build a complete Agentic RAG system using the Microsoft Agent Framework for .NET on Azure AI Foundry. Unlike classic RAG approaches that require manual orchestration and plugin wiring, the Agent Framework for .NET provides:

  • Built-in tools for Azure AI Search, web search, memory, and file search
  • Graph-based multi-agent workflows for planning and reasoning
  • First-class .NET SDKs and C# support
  • Production-ready hosting and deployment on Azure
No Python or Semantic Kernel required — this is a pure .NET solution.

3. The Four Agentic Pillars

1

🧠 Memory

Built-in Agent Memory: Managed long-term memory store with vector embeddings. Chat summary + user profile across sessions. Plus conversation threads for short-term context.

2

📋 Planning

Planner Agent in the workflow graph decomposes queries into sub-tasks with tool assignments. Uses GPT-4o with structured JSON output.

3

🔍 Reasoning

Reflection Agent evaluates evidence completeness, assigns confidence scores, identifies gaps, and triggers re-planning. Loop-back edge in the workflow graph (max 2×).

4

⚡ Caching

Redis Enterprise with HNSW vector index. Cosine similarity ≥ 0.92 = cache hit. Sub-100ms response for similar questions.

6. Microsoft Agent Framework — Deep Dive

What Is It?

The Microsoft Agent Framework is an open-source SDK (github.com/microsoft/agent-framework) for building AI agents that deploy to Azure AI Foundry. It provides:

  • Agent lifecycle — Create, configure, and manage agents with tools
  • Built-in tool catalog — AI Search, Web Search, Code Interpreter, File Search, Memory, Function Calling
  • Workflow engine — Graph-based multi-agent orchestration with edges, loops, and fan-out
  • Hosting adapter — One-line HTTP server for local dev and Foundry deployment
  • Observability — Tracing via App Insights, batch evaluation, prompt optimization

7. Built-In vs Custom Tools

🔍

AzureAISearchAgentTool

Built-in. Connects to Azure AI Search index. Supports hybrid (vector + keyword + semantic) queries. Just provide connection ID + index name. Returns results with citations.

🌐

WebSearchPreviewTool

Built-in. Real-time web search with citations. Zero setup — no Bing resource or API key needed. Configurable location and context size.

🧠

Agent Memory

Built-in. Managed long-term memory store with chat summaries and user profiles. Automatic embedding. Persists across sessions and devices.

📄

FileSearchTool

Built-in. Semantic + keyword search over uploaded files (PDF, DOCX, etc.) via vector stores. Auto-indexes uploaded content.

🏗️

FunctionTool (Document Intelligence)

Custom. Wraps Azure Document Intelligence for OCR, layout analysis, and table extraction from PDFs and images. Uses prebuilt-layout model.

🗄️

FunctionTool (SQL Server)

Custom. Read-only SELECT queries against SQL Server. Keyword blocklist prevents write/DDL operations. Results limited to 100 rows.

11. Planning Pillar — Planner Agent

agents/planner.py

import os
import json
from agent_framework.azure import AzureAIClient
from azure.identity.aio import DefaultAzureCredential

PLANNER_INSTRUCTIONS = """
You are a planning agent. Given a user question and any relevant memory context,
decompose it into an ordered list of steps. Each step must specify which tool to use.

Available tools:
- "search_knowledge_base": Search internal documents via Azure AI Search (hybrid)
- "query_sql": Query structured data in SQL Server (generate a SELECT query)
- "extract_document": Extract text from a specific PDF or image file path
- "web_search": Search the internet for real-time or external information
- "memory_recall": Retrieve relevant past memories about the user

Rules:
1. Start with memory_recall if the question might relate to past conversations
2. Prefer internal sources before web_search
3. For simple questions, 1-2 steps is fine
4. Always end with a synthesize step (toolToUse: "synthesize")

Respond with ONLY a JSON object: { "steps": [...] }
Each step: { "stepNumber": N, "description": "...", "toolToUse": "...", "toolInput": "..." }
"""


async def create_planner_agent(credential):
    """Create a dedicated planner agent."""
    client = AzureAIClient(
        project_endpoint=os.getenv("FOUNDRY_PROJECT_ENDPOINT"),
        model_deployment_name=os.getenv("FOUNDRY_MODEL_DEPLOYMENT_NAME"),
        credential=credential,
    )
    return await client.as_agent(
        name="RagPlanner",
        instructions=PLANNER_INSTRUCTIONS,
    ).__aenter__()


async def create_plan(planner_agent, question: str, memories: list[str] | None = None) -> list[dict]:
    """Generate an execution plan for the given question."""
    context = ""
    if memories:
        context = f"\n\nRelevant memories:\n" + "\n".join(memories)

    prompt = f"Question: {question}{context}"

    # Get non-streaming response for plan
    response = await planner_agent.run(prompt, stream=False)
    text = response.text

    # Parse JSON plan
    data = json.loads(text)
    steps = data.get("steps", data if isinstance(data, list) else [])
    return steps

import os
import json
from agent_framework.azure import AzureAIClient

REFLECTION_INSTRUCTIONS = """
You are a critical reasoning agent. Given a user's question and gathered evidence,
evaluate whether we have enough information for a complete, accurate answer.

Analyze:
1. Does the evidence directly answer the question?
2. Are there contradictions between sources?
3. Is any critical information missing?
4. How confident are you (0.0 to 1.0)?

Respond with ONLY a JSON object:
{
  "isComplete": true/false,
  "confidenceScore": 0.0-1.0,
  "gapAnalysis": "what's missing or null",
  "suggestedAction": "what to do next or null",
  "finalAnswer": "synthesised answer if isComplete=true, null otherwise"
}

If confidence >= 0.8, set isComplete=true and provide finalAnswer.
If confidence < 0.8, set isComplete=false and explain what's needed.
"""


async def create_reflection_agent(credential):
    """Create a dedicated reflection/reasoning agent."""
    client = AzureAIClient(
        project_endpoint=os.getenv("FOUNDRY_PROJECT_ENDPOINT"),
        model_deployment_name=os.getenv("FOUNDRY_MODEL_DEPLOYMENT_NAME"),
        credential=credential,
    )
    return await client.as_agent(
        name="RagReflector",
        instructions=REFLECTION_INSTRUCTIONS,
    ).__aenter__()


async def evaluate_evidence(reflection_agent, question: str, evidence: dict[str, str]) -> dict:
    """Evaluate gathered evidence and determine if answer is complete."""
    evidence_text = "\n\n".join(
        f"--- Source: {k} ---\n{v}" for k, v in evidence.items()
    )
    prompt = f"Question: {question}\n\nEvidence:\n{evidence_text}"

    response = await reflection_agent.run(prompt, stream=False)
    return json.loads(response.text)

13. Caching Pillar — Redis Semantic Cache

This is the one component we still implement custom — the Agent Framework doesn't include semantic caching. We use Redis Enterprise with RediSearch and HNSW vector index.

cache/semantic_cache.py

Python
import os
import json
import uuid
import numpy as np
import redis
from redis.commands.search.field import TextField, VectorField
from redis.commands.search.indexDefinition import IndexDefinition, IndexType
from redis.commands.search.query import Query
from openai import AzureOpenAI


class SemanticCache:
    def __init__(self):
        self.redis_client = redis.from_url(
            os.getenv("REDIS_URL"),
            password=os.getenv("REDIS_PASSWORD"),
            decode_responses=False,
        )
        self.embedding_client = AzureOpenAI(
            azure_endpoint=os.getenv("AZURE_OPENAI_ENDPOINT"),
            api_key=os.getenv("AZURE_OPENAI_KEY"),
            api_version="2024-10-21",
        )
        self.deployment = os.getenv("EMBEDDING_DEPLOYMENT")
        self.dims = int(os.getenv("EMBEDDING_DIMENSIONS", "3072"))
        self.threshold = float(os.getenv("CACHE_SIMILARITY_THRESHOLD", "0.92"))
        self.index_name = "semantic_cache_idx"
        self.prefix = "cache:"
        self._ensure_index()

    def _ensure_index(self):
        try:
            self.redis_client.ft(self.index_name).info()
        except redis.ResponseError:
            schema = (
                TextField("$.question", as_name="question"),
                TextField("$.answer", as_name="answer"),
                VectorField(
                    "$.embedding", "HNSW",
                    {"TYPE": "FLOAT32", "DIM": self.dims, "DISTANCE_METRIC": "COSINE"},
                    as_name="embedding",
                ),
            )
            self.redis_client.ft(self.index_name).create_index(
                schema, definition=IndexDefinition(prefix=[self.prefix], index_type=IndexType.JSON)
            )

    def _embed(self, text: str) -> list[float]:
        resp = self.embedding_client.embeddings.create(
            input=text, model=self.deployment, dimensions=self.dims
        )
        return resp.data[0].embedding

    def lookup(self, question: str) -> str | None:
        """Check if a similar question exists in cache."""
        embedding = self._embed(question)
        vec_bytes = np.array(embedding, dtype=np.float32).tobytes()

        q = (
            Query("(*)=>[KNN 1 @embedding $vec AS score]")
            .return_fields("question", "answer", "score")
            .sort_by("score")
            .dialect(2)
        )
        results = self.redis_client.ft(self.index_name).search(q, {"vec": vec_bytes})

        if results.total > 0:
            doc = results.docs[0]
            score = float(doc.score)
            similarity = 1.0 - score  # cosine distance → similarity
            if similarity >= self.threshold:
                return doc.answer.decode() if isinstance(doc.answer, bytes) else doc.answer
        return None

    def store(self, question: str, answer: str):
        """Store a Q&A pair with its embedding."""
        embedding = self._embed(question)
        key = f"{self.prefix}{uuid.uuid4().hex}"
        self.redis_client.json().set(key, "$", {
            "question": question,
            "answer": answer,
            "embedding": embedding,
        })

def extract_document(file_path: str) -> str:
    """Extract text and tables from a PDF or image using Azure Document Intelligence.

    Args:
        file_path: Full path to the PDF or image file to extract text from.

    Returns:
        Extracted text content including page text and structured tables.
    """
    with open(file_path, "rb") as f:
        poller = client.begin_analyze_document(
            "prebuilt-layout",
            analyze_request=f,
            content_type="application/octet-stream",
        )
    result = poller.result()

    lines = []
    # Extract page text
    if result.pages:
        for page in result.pages:
            lines.append(f"--- Page {page.page_number} ---")
            if page.lines:
                for line in page.lines:
                    lines.append(line.content)

    # Extract tables
    if result.tables:
        for i, table in enumerate(result.tables):
            lines.append(f"\n--- Table {i + 1} ---")
            rows: dict[int, dict[int, str]] = {}
            for cell in table.cells:
                rows.setdefault(cell.row_index, {})[cell.column_index] = cell.content
            for row_idx in sorted(rows):
                vals = [rows[row_idx].get(c, "") for c in range(table.column_count)]
                lines.append(" | ".join(vals))

    return "\n".join(lines)

tools/sql_server.py — Read-Only SQL

Security: Only SELECT queries permitted. INSERT, UPDATE, DELETE, DROP, EXEC, and all DDL/DML write operations are blocked. Results limited to 100 rows. 
Python
import os
import pyodbc

BLOCKED_KEYWORDS = {
    "INSERT", "UPDATE", "DELETE", "DROP", "ALTER", "CREATE",
    "TRUNCATE", "EXEC", "EXECUTE", "MERGE", "GRANT", "REVOKE", "xp_", "sp_",
}


def query_sql(query: str) -> str:
    """Execute a read-only SELECT query against SQL Server and return results.

    Args:
        query: A SQL SELECT query to run. Only SELECT statements are allowed.

    Returns:
        Query results formatted as a pipe-delimited table, or an error message.
    """
    trimmed = query.strip()
    first_word = trimmed.split()[0].upper() if trimmed else ""

    if first_word != "SELECT":
        return "Error: Only SELECT queries are allowed."

    upper_query = query.upper()
    for kw in BLOCKED_KEYWORDS:
        if kw.upper() in upper_query:
            return f"Error: '{kw}' operations are not permitted."

    conn = pyodbc.connect(os.getenv("SQL_CONNECTION_STRING"))
    cursor = conn.cursor()
    cursor.execute(trimmed)

    columns = [desc[0] for desc in cursor.description]
    lines = [" | ".join(columns)]
    lines.append("-" * len(lines[0]))

    row_count = 0
    for row in cursor.fetchall():
        if row_count >= 100:
            break
        lines.append(" | ".join(str(v) if v is not None else "NULL" for v in row))
        row_count += 1

    cursor.close()
    conn.close()
    return "\n".join(lines)

from tools.sql_server import query_sql
from cache.semantic_cache import SemanticCache

logger = logging.getLogger(__name__)

MAX_REPLAN_ATTEMPTS = 2


async def build_rag_workflow():
    """Build the complete Agentic RAG workflow graph."""
    credential = DefaultAzureCredential()
    cache = SemanticCache()

    # ─── Set up project client for AI Search connection ───
    project_client = AIProjectClient(
        endpoint=os.getenv("FOUNDRY_PROJECT_ENDPOINT"),
        credential=credential,
    )
    search_connection = project_client.connections.get(
        os.getenv("AZURE_AI_SEARCH_CONNECTION_NAME")
    )

    # ─── Built-in Tools ───
    ai_search_tool = AzureAISearchToolDefinition(
        azure_ai_search=AzureAISearchToolResource(indexes=[
            AISearchIndexResource(
                index_connection_id=search_connection.id,
                index_name=os.getenv("AI_SEARCH_INDEX_NAME"),
                query_type=AzureAISearchQueryType.VECTOR_SEMANTIC_HYBRID,
            ),
        ])
    )

    web_search_tool = WebSearchPreviewTool()  # Zero config!

    # ─── Custom FunctionTools ───
    custom_functions = FunctionTool([extract_document, query_sql])
    toolset = ToolSet()
    toolset.add(custom_functions)

    # ═══════════════════════════════════════════
    # CREATE THREE SPECIALISED AGENTS
    # ═══════════════════════════════════════════

    # 1. Planner Agent — decomposes queries
    planner_client = AzureAIClient(
        project_endpoint=os.getenv("FOUNDRY_PROJECT_ENDPOINT"),
        model_deployment_name=os.getenv("FOUNDRY_MODEL_DEPLOYMENT_NAME"),
        credential=credential,
    )
    planner_agent = await planner_client.as_agent(
        name="RagPlanner",
        instructions=PLANNER_INSTRUCTIONS,
    ).__aenter__()

    # 2. Executor Agent — has ALL tools + memory
    executor_client = AzureAIClient(
        project_endpoint=os.getenv("FOUNDRY_PROJECT_ENDPOINT"),
        model_deployment_name=os.getenv("FOUNDRY_MODEL_DEPLOYMENT_NAME"),
        credential=credential,
    )
    executor_agent = await executor_client.as_agent(
        name="RagExecutor",
        instructions="""You are an intelligent RAG executor. Use your tools to gather evidence:
- Use search_knowledge_base (AI Search) for internal documents
- Use web_search for real-time internet information
- Use extract_document for specific PDF/image files
- Use query_sql for structured database queries
Gather comprehensive evidence and return it clearly organized.""",
        tools=[ai_search_tool, web_search_tool],
        toolset=toolset,
        memory_store="rag-memory",
    ).__aenter__()

    # 3. Reflection Agent — evaluates evidence
    reflection_client = AzureAIClient(
        project_endpoint=os.getenv("FOUNDRY_PROJECT_ENDPOINT"),
        model_deployment_name=os.getenv("FOUNDRY_MODEL_DEPLOYMENT_NAME"),
        credential=credential,
    )
    reflection_agent = await reflection_client.as_agent(
        name="RagReflector",
        instructions=REFLECTION_INSTRUCTIONS,
    ).__aenter__()

    # ═══════════════════════════════════════════
    # ORCHESTRATION FUNCTION
    # ═══════════════════════════════════════════

    async def ask(question: str, user_id: str = "default") -> dict:
        """Full agentic RAG pipeline: Cache → Plan → Execute → Reflect → Re-plan."""

        # Step 1: Semantic Cache
        cached = cache.lookup(question)
        if cached:
            logger.info("Cache HIT")
            return {"answer": cached, "from_cache": True}

        # Step 2: Plan
        logger.info("Creating execution plan...")
        plan_response = await planner_agent.run(
            f"Question: {question}", stream=False
        )
        plan = json.loads(plan_response.text).get("steps", [])
        logger.info("Plan has %d steps", len(plan))

        # Step 3-5: Execute + Reflect loop
        evidence = {}
        final_answer = None

        for attempt in range(MAX_REPLAN_ATTEMPTS + 1):
            # Execute — send plan to executor agent (which has all tools)
            exec_prompt = f"""Execute this plan to answer: {question}

Plan:
{json.dumps(plan, indent=2)}

Gather evidence from each step and return organized results."""

            exec_response = await executor_agent.run(exec_prompt, stream=False)
            evidence[f"Attempt {attempt + 1}"] = exec_response.text

            # Reflect
            logger.info("Reflecting (attempt %d/%d)...", attempt + 1, MAX_REPLAN_ATTEMPTS + 1)
            reflect_prompt = f"Question: {question}\n\nEvidence:\n{exec_response.text}"
            reflect_response = await reflection_agent.run(reflect_prompt, stream=False)
            reasoning = json.loads(reflect_response.text)

            logger.info(
                "Confidence: %.2f | Complete: %s",
                reasoning["confidenceScore"], reasoning["isComplete"]
            )

            if reasoning["isComplete"] and reasoning.get("finalAnswer"):
                final_answer = reasoning["finalAnswer"]
                break

            # Re-plan with gap info
            if attempt < MAX_REPLAN_ATTEMPTS and reasoning.get("suggestedAction"):
                logger.info("Re-planning — gap: %s", reasoning.get("gapAnalysis"))
                replan_prompt = (
                    f"Question: {question}\n"
                    f"Gap: {reasoning.get('gapAnalysis')}\n"
                    f"Suggested: {reasoning.get('suggestedAction')}"
                )
                replan_response = await planner_agent.run(replan_prompt, stream=False)
                plan = json.loads(replan_response.text).get("steps", [])

        if not final_answer:
            final_answer = "Unable to find a satisfactory answer after multiple attempts."

        # Step 6: Cache the result
        cache.store(question, final_answer)
        logger.info("Answer cached.")

        return {"answer": final_answer, "from_cache": False}

    return ask

from openai import AzureOpenAI
from tools.doc_intelligence import extract_document


def create_index(index_client: SearchIndexClient, index_name: str):
    """Create the Azure AI Search index with vector support."""
    fields = [
        SimpleField(name="id", type=SearchFieldDataType.String, key=True),
        SearchableField(name="content", analyzer_name="en.microsoft"),
        SimpleField(name="source", type=SearchFieldDataType.String, filterable=True),
        SimpleField(name="title", type=SearchFieldDataType.String),
        SearchField(
            name="content_vector",
            type=SearchFieldDataType.Collection(SearchFieldDataType.Single),
            searchable=True, vector_search_dimensions=3072,
            vector_search_profile_name="hnsw-profile",
        ),
    ]
    vector_search = VectorSearch(
        algorithms=[HnswAlgorithmConfiguration(name="hnsw-config")],
        profiles=[VectorSearchProfile(name="hnsw-profile", algorithm_configuration_name="hnsw-config")],
    )
    index = SearchIndex(name=index_name, fields=fields, vector_search=vector_search)
    index_client.create_or_update_index(index)


def chunk_text(text: str, size: int = 1000, overlap: int = 200) -> list[str]:
    chunks, start = [], 0
    while start < len(text):
        chunks.append(text[start : start + size])
        start += size - overlap
    return chunks


def ingest_file(file_path: str):
    """Extract text from a file, chunk, embed, and upload to Azure AI Search."""
    oai = AzureOpenAI(
        azure_endpoint=os.getenv("AZURE_OPENAI_ENDPOINT"),
        api_key=os.getenv("AZURE_OPENAI_KEY"),
        api_version="2024-10-21",
    )
    search_cred = AzureKeyCredential(os.getenv("SEARCH_API_KEY"))
    index_name = os.getenv("AI_SEARCH_INDEX_NAME")

    # Create index if needed
    idx_client = SearchIndexClient(os.getenv("SEARCH_ENDPOINT"), search_cred)
    create_index(idx_client, index_name)

    # Extract + chunk
    text = extract_document(file_path)
    chunks = chunk_text(text)

    # Embed + upload
    docs = []
    for i, chunk in enumerate(chunks):
        resp = oai.embeddings.create(
            input=chunk, model=os.getenv("EMBEDDING_DEPLOYMENT"), dimensions=3072
        )
        docs.append({
            "id": uuid.uuid4().hex,
            "content": chunk,
            "source": file_path,
            "title": f"{os.path.basename(file_path)} - chunk {i + 1}",
            "content_vector": resp.data[0].embedding,
        })

    search_client = SearchClient(os.getenv("SEARCH_ENDPOINT"), index_name, search_cred)
    for batch in [docs[i:i+100] for i in range(0, len(docs), 100)]:
        search_client.upload_documents(batch)

    print(f"Ingested {len(docs)} chunks from {file_path}")
